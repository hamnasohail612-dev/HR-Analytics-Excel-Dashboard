# HR & Payroll Analytics Dashboard (200 Workers)

An automated corporate HR project built in Microsoft Excel to analyze headcount distribution, payroll budget, and employee performance ratings.

## 📊 Key Highlights of the Project
- **Master Dataset:** Structured directory tracking 200 employee records with Department, Role, Salary, Status, and Performance Ratings.
- **Dynamic Dashboard:** Formatted dynamic charts tracking **$16.03M** total payroll allocation.
- **Advanced Summaries:** Implemented custom Pivot Tables and interactive data slicing (Status & Department).

## 🗂️ Sheet Structure
1. `Master_Data`: Contains the raw employee matrix with verified formatting.
2. `Dashboard`: Executive view hosting analytical breakdown charts and filtering slicers.
